Was ist Plainly?

Plainly ist eine Cross-Platform App zur Vereinfachung von Texten mithilfe der OpenAI API.
Verstehst Du manchmal komplizierte Texte nicht auf Anhieb? Mit Plainly ist das kein Problem mehr! Unsere App ermöglicht es Dir, Texte ganz einfach einzuscannen und in eine leicht verständliche Sprache umzuwandeln. Aber das ist noch nicht alles! Wir bieten Dir auch die Möglichkeit, diese vereinfachten Texte in eine Vielzahl von Sprachen zu übersetzen.

Mögliche Vereinfachungsoptionen

Mit Plainly können Dokumente mit der Kamera eingescannt werden, über die Fotogalerie oder als PDF hochgeladen werden. Alternativ können die Nutzer auch ein Freitext zur Vereinfachung eingeben. Der Text wird auf dem Bild oder Dokument eingelesen und über die OpenAI API in einfachere Sprache vereinfacht.
